Marble Game

Setup and goal

There are C identical tubes, each holding max H marbles.

Marbles have N distinct colors with exactly H marbles per color (total: Nâ€¢H marbles).

Initially, N tubes are randomly filled with all marbles and K = C - N tubes are empty.

Goal: Arrange marbles so each tube contains H marbles of a single color with all tubes filled to capacity.

Moving rules

Move top marble from tube A to tube B if:

  • Tube B is empty, OR
  • Top marble of B matches the color of the moved marble

Movement options (configurable):

  • Move all consecutive same-colored marbles at once, OR
  • Move only one marble at a time

Optional mode: Only the top marble is initially visible. Lower marbles reveal when becoming top.
